Programming Language:
C++

=========================================================
PROJECT DESCRIPTION
=========================================================

Library Book Records System is a console-based application
developed using C++. The system allows administrators to
manage books and customers while customers can search,
borrow, return and reserve books.

The project demonstrates Object-Oriented Programming (OOP),
Dynamic Data Structures, Searching Algorithms, Sorting
Algorithms, File Handling and Exception Handling.
